# what is List in python ? 

1. list is a inbuilt data structure in python which is used to store multiple items in a single variable.

2. list is ordered, changeable and allows duplicate values.

3. list is mutable, which means you can change the values of a list after it has been created.

4. list is defined by enclosing the items in square brackets [].

5. list is also defined as a list constructor which is a built-in function that creates a list from an iterable.

6. list can contain items of different data types, including other lists.

7. list is a versatile data structure that can be used for various purposes, such as storing a collection of items, performing operations on the items, and more.

8. list is a powerful data structure that provides various methods and functions to manipulate the items in the list, such as append(), remove(), sort(), extend(), insert(),  etc.

9. list is used to slice and index from 0 the items in the list, which allows you to access specific items or a range of items in the list.

  **examples of defined list in python :**

```python
# defining a list of integers
my_list = [1, 2, 3, 4, 5]
print(my_list)
# defining a list of strings
my_list = ["apple", "banana", "cherry"]
print(my_list)
# defining a list of mixed data types
my_list = [1, "hello", 3.14, True]
print(my_list)
# defining a list of lists
my_list = [[1, 2, 3], ["a", "b", "c"]]
print(my_list)

# defined list as constructor
my_list = list((1, 2, 3, 4, 5))
print(my_list)
my_list = list(("apple", "banana", "cherry"))
print(my_list)
my_list = list((1, "hello", 3.14, True))
print(my_list)
my_list = list(([1, 2, 3], ["a", "b", "c"]))
print(my_list)



```
